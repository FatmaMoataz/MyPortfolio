## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/FatmaMoataz/portfolio.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
